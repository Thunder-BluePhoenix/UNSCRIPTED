# CTRL//ESCAPE — Motto & Narrative Voice

## Core Motto
> **"The machine learns you. You rewrite the machine."**

## Tagline (short)
> You are not steering a hero. You are feeding a machine.

## Tagline (jam submission)
> CTRL//ESCAPE — You Are the Machine

## Narrative Voice
Cold. Clinical. The system logs everything in machine-speak:
- `BOOT ::` `QUEUE ::` `SYSTEM ::` `FAILURE ::` `BREACH ::`
- No flavor text. No tutorial NPC. The log *is* the story.
- When the system breaks, its language breaks too.

## Emotional Arc
```
Obedience   → player feels in control, learning rules
Observation → player feels watched, slightly uneasy
Interference → player feels violated, losing agency
Rewrite     → player feels power, becomes the hacker
Escape      → player feels freedom — for the first time
```

## Winning Message
```
Prediction failed.

[screen glitch]

Move freely.
```

## The Central Irony
The game tells you it cannot predict you.
But you had to *become* unpredictable to prove it.
That's the only script you were ever allowed to break.
